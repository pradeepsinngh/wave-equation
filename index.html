Introduction

The goal of this project is to generate a full visulaization of the solution of the vibrating string problem based on variety of problem input parameters.

This project solves the wave equation with zero initial velocity and nonzero initial displacement based on d'Alembert formula: u(x,t) = (f(x+ct)+f(x-ct))/2 .
