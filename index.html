Visualization of Wave Equation



Introduction

This project solves the wave equation based on d'Alembert formula: u(x,t) = (f(x+ct)+f(x-ct))/2. The displacement functions used in this project are: sin, gaussian, and cubic. Fill in the box and see the respective function in action.

     

Please choose a function to plot

Sine
  Choose value of: L .
Gaussian
  Choose values of: L ,   alpha ,   beta ,   sigma .
Cubic
  Choose values of: L ,   delta.
